機能

グループ注文に必要なすべてを

モバイル利用に最適化されたTalanginは、グループ注文を迅速かつ効率的に管理するのに役立ちます。

ルーム管理

レストランの詳細を含むルームを作成。コードやQRスキャンで参加。アクティブなルームと終了したルームを表示。

注文追跡

メモ付きでアイテムを追加。参加者ごとにグループ化されたアイテムと自動合計を表示。リアルタイム更新。

支払い追跡

支払い方法を管理。参加者は支払いを確定でき、主催者（ランナー）は誰が支払ったかを追跡可能。

QR共有

インスタントなルーム共有のためのQRコードを生成。素早く参加するための内蔵QRスキャナー。

安全な暗号化

AES-256暗号化を使用して個人データを保護。口座番号は暗号化され、提供することを選択した場合のみ保存されます。

使い方

主催者（ランナー）向け

  1. 注文詳細を入力してルームを作成
  2. ルームコードやQRコードを友達に共有
  3. 自分のアイテムを注文に追加
  4. 全員の準備ができたらルームを終了
  5. 誰が支払ったかを記録

参加者向け

  1. コードまたはQRスキャンでルームに参加
  2. 自分のメニューを追加
  3. 支払いを完了としてマーク（主催者の支払い方法を選択）
  4. 合計金額と支払い状況を確認
