Features

Everything you need for group orders

Optimized for mobile use, Talangin helps you manage group orders quickly and efficiently.

Room Management

Create rooms with restaurant details. Join via code or QR scan. View active and closed rooms.

Order Tracking

Add items with notes. View grouped items by participant with automatic totals. Real-time updates.

Payment Tracking

Manage payment methods. Participants can confirm payments. Runners can track who has paid.

QR Sharing

Generate QR codes for instant room sharing. Built-in QR scanner to join rooms quickly.

How it works

For the Runner (Order Creator)

  1. Create a room with order details
  2. Share room code or QR code with friends
  3. Add your own items to the order
  4. Close the room when everyone is ready
  5. Record who has paid

For Participants (Joiners)

  1. Join room via code or QR scan
  2. Add your meal items
  3. Mark payment as done (select runner's payment method)
  4. View your total and payment status
