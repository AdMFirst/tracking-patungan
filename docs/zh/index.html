功能亮点

拼单点餐，这一款就够了

专为手机优化，助你快速高效管理多人订单。

房间管理

创建包含餐厅信息的房间。通过代码或扫码加入。查看进行中和已结束的拼单。

订单追踪

添加餐品和备注。按人归类显示，自动计算总价。实时同步更新。

支付追踪

管理收款方式。参与者确认付款，发起人（团长）追踪谁已付款。

二维码分享

生成二维码快速分享房间。内置扫码器，一键加入拼单。

安全加密

采用 AES-256 加密保护你的隐私。银行账号仅在你选择提供时加密存储。

使用流程

发起人（团长）

  1. 创建房间，填写餐厅信息
  2. 分享房间号或二维码给好友
  3. 添加你自己的餐品
  4. 大家点完后锁定房间
  5. 记录谁已经付款

参与者

  1. 扫码或输入代码加入房间
  2. 添加你想吃的餐品
  3. 标记“已付款”（选择团长的收款方式）
  4. 查看你的应付金额和支付状态
